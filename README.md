# Online Food Ordering System - SDLC using GitHub

## Description
This project demonstrates SDLC implementation using GitHub, including version control, CI/CD, testing, and security.

## Features
- GitHub Actions CI/CD pipeline
- Automated testing using Pytest
- Code quality checks using Flake8
- Security analysis using Bandit

## Project Structure
- src/ → Source code
- tests/ → Test cases
- .github/workflows → CI/CD pipeline
